# GameAddiction
## Contexte :
Les données viennent de différents questionnaires addressés aux gamers dans le contexte de la recherche. Ces questionnaires sont collectés par des experts dans le but d'explorer les impacts des jeux vidéo sur la santé mentale et le comportement. Ces recherches sont uniques et non reproduite pour le moment. Leur qualité n'est donc pas parfaite.

## Description :
Les données sont structurées en tableaux de valeurs. Les données incluent l'âge, le genre, les habitudes de jeu, les heures d'exercice quotidien, le niveau de stress, etc.

## But : 
Je veux démontrer que l'addiction aux jeux vidéo n'existe pas en partant des différents problèmes liés à l'addiction et montrer que ceux-cis ne se retrouvent pas chez les gros joueurs. Je veux explorer cette hypothèse et la nunacer potentiellement.

## Références :
Ce sujet a été abordé par l'Observatoire de la Parentalité & de l'Éducation Numérique dans le but de montrer que l'addiction aux jeux vidéo est un vrai trouble et n'est pas simplement le fait de jouer beaucoup à des jeux vidéo.
